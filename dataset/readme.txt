📦 Dataset Information

This folder contains a small sample of the CarDD dataset for demo/testing purposes.

✅ Sample includes:
- YOLO-formatted images 

📂 Full Dataset (COCO Format):
- Google Drive Link: https://drive.google.com/drive/u/0/folders/1LkND5biBr3IBxsbkn_7z_mcLgqFQPach

🔄 Conversion:
- Use the script in `src/convert_dataset.py` to convert COCO to YOLO format
- Output will be organized as: /train, /val, /test with images and labels

📝 Dataset YAML used: CarDD_6class_dataset.yaml
